# DWSC

**DWSC: Deep Clustering via Weighted k-Subspace Network**

This repo hosts the code for paper "Deep Clustering via Weighted k-Subspace Network".

## Requirements

```
tensorflow-gpu==1.12.3
keras==2.2.4
scikit-learn==0.22.2
numpy
```

## Train

A simple way is to run train.py

```
python train.py
```
